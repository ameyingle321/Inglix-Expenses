# Expense Tracker Application

A finance management application featuring expense tracking, budgeting functionality, and data visualization for personal finance management.

![Screenshot](./screenshot.png)

## Features

- Track daily expenses and income
- Budget management
- Data visualization with charts
- User authentication

## Tech Stack

- React with JavaScript
- Vite
- Node.js / Express (backend)
- MongoDB
- Chart.js
- Authentication

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```
